# d3源码学习计划

## 学习资源

说道学习资源，那么第一步肯定是看源码啦，官方还很贴心的把每个api的源码和例子附在旁边，不学都辜负了别人的好心。

学习策略：源码和例子同时学习

## 源码学习
按照[官方文档](https://github.com/d3/d3/wiki/Tutorials) 中的api顺序，每天学几个。
